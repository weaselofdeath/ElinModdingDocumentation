<h2 style="text-align:center;"><strong>What is this?</strong></h2>
<p style="text-align:center;">This is documentation for the code from the game <a target="_blank" rel="noopener noreferrer" href="https://store.steampowered.com/app/2135150/Elin/">Elin</a>.&nbsp;</p>
<p style="text-align:center;">It's intended to be used as a reference for creating mods using BepInEx.</p>
<hr>
<h2 style="text-align:center;"><strong>Why?</strong></h2>
<p style="text-align:center;">I wanted an easy reference to look at while modding and figured it would be helpful to others.</p>
<p style="text-align:center;">Hopefully, this will help people make mods.</p>
